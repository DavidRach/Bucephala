## `Bucephala`: Being Everything, Everywhere, All at Once.

This is a work-in-progress, that shoul not be installed by any sane individual in it's current form. **Ye hath been warned**

### Installation

``` r
if(!require("remotes")) install.packages("remotes")

remotes::install_github("https://github.com/DavidRach/Bucephala")

library(Bucephala)

```
